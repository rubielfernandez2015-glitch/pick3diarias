🎯 Modo de Resultado

    

      Elige cómo se obtiene el número ganador cada día. Aplica para todo el equipo.
    

    

      
      
    

    

      

        ✋ El banquero toca Buscar Resultado cuando lo desee y el sistema trae el número de Florida Lottery automáticamente. También puede ingresarlo a mano. El cierre se ejecuta al confirmar.

        Recomendado para verificar que todas las jugadas subieron antes de cerrar.
      

    

    

      

        🤖 El sistema obtiene el número de Florida Lottery automáticamente a la hora configurada y realiza el cierre.
      

      

        

          
          
        

        

          
          
        

      

      
⏰ Hora de Cuba (UTC-5). El sistema verificará cada minuto.

      

        ⚠️ Asegúrate de que todas las jugadas estén sincronizadas antes de la hora del cierre automático.

⚠️ Límite Global de Alerta

    
Cuando un número supera el límite se activa la alerta. Elige cómo calcularlo:

    

      
      
    

    

      
Monto máximo permitido por número en el día.

      

    

    

      
El sistema calcula: Neto × % ÷ 80 · Neto = total recogido × (1 - comisión%)

      

        
        %
        
      

      
70% = límite conservador · 100% = límite máximo
